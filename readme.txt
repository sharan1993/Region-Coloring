Hopfield Network Implementation for Coloring Maps

Contents of Archive:
1. sharanr-takehome3.pdf - document containing results
2. readme.txt
3. Software Directory

Contents of Software Directory:
1. FeedInputM1 - function that initialize hopfield net for Map 1
2. FeedInputM2 - function that initialize hopfield net for Map 2
3. WeightMap1 - function that calculates weight matrix and bias for Map 1 constraints
4. WeightMap2 - function that calculates weight matrix and bias for Map 2 constraints
5. script - executes all functions and displays output state, state transition, and energy plot (execute this)
